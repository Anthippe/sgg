-- Οδηγίες εγκατάστασης SGG σε macOS by @gsiros --

- Οι οδηγίες δοκιμάστηκαν σε περιβάλλον macOS Catalina 10.15.7 (19H15), δεν γνωρίζω αν δουλεύουν για παλαιότερες εκδόσεις macOS.

- ΑΠΟΠΟΙΗΤΙΚΟ ΕΥΘΥΝΩΝ: Δεν είμαι υπεύθυνος για οποιοδήποτε πρόβλημα που μπορεί να προκύψει από την διαδικασία εγκατάστασης στο περιβάλλον σας. Τα παρακάτω βήματα δούλεψαν για το δικό μου περιβάλλον και λογικά θα δουλεύουν για το δικό σας. Είμαι φοιτητής και όχι ο επίσημος developer των βιβλιοθηκών. Ευχαριστώ!

- ΟΔΗΓΙΕΣ:

Από περιβάλλον τερματικού, πλοηγούμαστε στον φάκελο που θέλουμε να κατεβάσουμε την βιβλιοθήκη και εκτελούμε μια-μια τις παρακάτω εντολές με αντιγραφή/επικόλληση και enter (στις περισσότερες θα χρεαστεί να περιμένουμε να γίνει η εγκατάσταση των πακέτων):

ΣΗΜΕΙΩΣΗ: Υπάρχει περίπτωση κατά την εκτέλεση κάποιας από τις παρακάτω εντολές να ζητηθεί από την κονσόλα ο κωδικός χρήστη (Password: ). Πληκτρολογούμε τον κωδικό. Αν και δεν φαίνονται τα ψηφία που πληκτρολογούμε, απλά τα κρύβει το τερματικό. Πατάμε enter.

0) git clone https://github.com/cgaueb/sgg.git
1) cd sgg
2) chmod +x SDL2_patch_macOS.sh
3) chmod +x build_sgg_macOS.sh
4) chmod +x build_demo_macOS.sh
5) /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)"
6) brew install sdl2
7) brew install sdl2_mixer
8) brew install glew
9) brew install freetype
10) ./SDL2_patch_macOS.sh
11) ./build_sgg_macOS.sh
12) ./build_demo_macOS.sh
13) cd bin
14) open .

Σε αυτό το σημείο έχει ανοίξει ο φάκελος bin με το Finder και βλέπουμε το demo και demod executable. Διπλό κλικ σε demo ή demod και ελέγχουμε αν όλα τρέχουν ομαλά.
